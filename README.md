# 🏥 Noor Hospital Website

Welcome to the official repository for **Noor Hospital** website.
This project represents the digital presence of the hospital and provides essential information about facilities, accessibility, services, and payment options.

---

## 📌 Project Overview

The Noor Hospital website aims to:

* Provide clear information about hospital facilities
* Ensure accessibility details are visible to all visitors
* Highlight available services and amenities
* Offer transparent payment options
* Improve patient convenience and trust
